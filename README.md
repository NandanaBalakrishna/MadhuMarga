Madhu Marga App
Madhu Marga is a smart apiary management Android application developed using Kotlin and Jetpack Compose. The application helps beekeepers track hive health, monitor environmental telemetry, and follow seasonal flora cycles through a modern, cyber-themed mobile platform.

Problem Statement
Beekeepers face difficulties in tracking hive health, queen productivity, and environmental changes manually. Madhu Marga was developed to digitize apiary records, providing AI-driven suggestions and real-time data tracking to prevent hive loss and improve honey production.

Features
User Registration and Login,
Central Apiary Registry,
Unit Telemetry (Temp/Humidity Tracking),
AI-Driven Beekeeping Suggestions,
Historical Log Management,
Monthly Flora Distribution Calendar

Technologies Used
